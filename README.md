# ILMIA - Medical AI Landing Page

A modern landing page for ILMIA (Ilmia Tech), a medical AI company.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Deployment

Build and deploy to Firebase:

```bash
npm run deploy
```

Or build only:

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Firebase Hosting
