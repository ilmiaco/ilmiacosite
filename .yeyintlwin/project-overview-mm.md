# ILMIA (イルミアテック) - ပရောဂျက်အကြောင်း

## ပရောဂျက်အကျဉ်းချုပ်

ILMIA (イルミアテック / အီလ်မီယာတက်ခ်) သည် ဂျပန်နိုင်ငံ၊ အိုဆာကာမြို့အခြေစိုက် **ဆေးဘက်ဆိုင်ရာ AI ပလက်ဖောင်း** ကုမ္ပဏီတစ်ခုဖြစ်သည်။ ဤဝဘ်ဆိုက်သည် ကုမ္ပဏီ၏ ထုတ်ကုန်နှစ်ခုဖြစ်သော **eDenshi Karte AI (အီလက်ထရောနစ် ဆေးမှတ်တမ်း AI)** နှင့် **Receipt AI (လက်ခံဖြတ်ပိုင်း AI)** တို့ကို မိတ်ဆက်ရန် တည်ဆောက်ထားသော Landing Page / Marketing Website ဖြစ်သည်။

## ထုတ်ကုန်နှစ်ခု

### 1. 電子カルテAI (Electronic Medical Record AI)
- အသံမှတ်ယူစနစ် (Voice Recognition) ဖြင့် ဆရာဝန်နှင့် လူနာ၏ စကားပြောဆိုမှုကို အလိုအလျောက် စာသားအဖြစ် ပြောင်းလဲပေးသည်
- AI ဖြင့် ဆေးမှတ်တမ်းကို အလိုအလျောက် ရေးသားပေးသည်
- ဆရာဝန်များ၏ "ရေးသားချိန်" ကို "စစ်ဆေးချိန်" အဖြစ် ပြောင်းလဲပေးသည်

### 2. レセプトAI (Receipt/Medical Billing AI)
- ဆေးဖိုးတောင်းခံလွှာ တင်ပြမှုမတိုင်မီ လွတ်ကျန်မှုများကို AI ဖြင့် စစ်ဆေးပေးသည်
- ရောဂါအမည် လွတ်ကျန်မှု၊ တွက်ချက်မှု လွတ်ကျန်မှုများကို အလိုအလျောက် ရှာဖွေပေးသည်
- နောက်ဆုံးပေါ် စည်းမျဉ်းများနှင့် အလိုအလျောက် ညှိနှိုင်းပေးသည်
- ဝင်ငွေ ဆုံးရှုံးမှုကို ကာကွယ်ပေးသည်

## အသုံးပြုထားသော နည်းပညာများ

### Frontend Framework & Language
- **React 18** — UI တည်ဆောက်ရန်
- **TypeScript** — Type-safe ကုဒ်ရေးသားရန်
- **Vite 6** — Build tool နှင့် Development server

### Styling
- **Tailwind CSS 4** — Utility-first CSS framework
- **tw-animate-css** — Animation များအတွက်

### UI Component Libraries
- **Radix UI** — Headless UI components (Accordion, Dialog, Dropdown, Tabs, Tooltip စသည်)
- **shadcn/ui pattern** — Radix UI အပေါ်တွင် တည်ဆောက်ထားသော UI components
- **MUI (Material UI) 7** — Icons နှင့် အချို့ components
- **Lucide React** — Icon library

### Animation & Interaction
- **Motion (Framer Motion) 12** — Animation library
- **Embla Carousel** — Carousel/slider component
- **react-slick** — Slider component
- **Vaul** — Drawer component
- **react-dnd** — Drag and drop

### Form & Data
- **react-hook-form** — Form management
- **Formspree** — Form submission backend (ပြင်ပဝန်ဆောင်မှု)
- **Recharts** — Charts/graphs
- **date-fns** — Date utility
- **react-day-picker** — Calendar/date picker
- **Sonner** — Toast notifications
- **cmdk** — Command palette

### Backend & Hosting
- **Firebase Hosting** — Static site hosting
- **Firebase Analytics** — Web analytics
- **Firebase** — Backend platform

### Build & Development Tools
- **Vite** — Bundler နှင့် dev server
- **PostCSS** — CSS processing
- **Sharp** — Image optimization (build scripts)
- **@tailwindcss/vite** — Tailwind CSS Vite plugin

### SEO & Web Standards
- **PWA (Progressive Web App)** — manifest.json ဖြင့် ပံ့ပိုးထားသည်
- **Schema.org structured data** — SEO metadata
- **Dublin Core metadata** — Professional citations
- **OpenSearch** — Site search protocol
- **robots.txt, sitemap.xml** — Search engine optimization
- **Security headers** — X-Frame-Options, CSP, XSS protection

### Project Architecture
- **Single Page Application (SPA)** — React SPA, routing မသုံးဘဲ anchor links ဖြင့် section navigation
- **Component-based architecture** — Sections များကို သီးခြား components အဖြစ် ခွဲခြားထားသည်
- **Path alias** — `@/` prefix ဖြင့် src directory ကို ညွှန်းသည်

## ဝဘ်ဆိုက်၏ ဖွဲ့စည်းပုံ

ဤဝဘ်ဆိုက်သည် **Single Page** ဒီဇိုင်းဖြစ်ပြီး routing library မသုံးပါ။ Section များကို anchor links (`#emr-ai`, `#receipt-ai`, `#pricing` စသည်) ဖြင့် navigate လုပ်သည်။ ဘေးညွှန် URL (`/index.html`, `/home`) များကို `/` သို့ 301 redirect လုပ်ထားသည်။
